# Libro en Markdown

En esta carpeta se incluyen los capítulos del libro en un formato accesible desde GitHub.  
Pueden leerse directamente como páginas web y también editarse para propuestas de mejora, correcciones o nuevos contenidos.

Esta versión en Markdown existe para que cualquier persona pueda:

- Revisar el texto sin necesidad de LaTeX.  
- Proponer cambios mediante *pull requests*.  
- Crear capítulos adicionales o comentarios críticos.  
- Facilitar la lectura en dispositivos móviles o navegadores.

La estructura de los capítulos respeta el orden del libro, y cada archivo es independiente para facilitar su edición.  
Los cambios aquí eventualmente se integrarán en la versión PDF y LaTeX.
